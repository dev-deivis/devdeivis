# _devdeivis | Link Bio

Página de links estilo terminal para Instagram bio.

## 🚀 Deploy en Vercel (3 pasos)

1. Sube esta carpeta a un repo en GitHub
2. Entra a [vercel.com](https://vercel.com) → **New Project** → importa el repo
3. Vercel detecta Vite automáticamente → click **Deploy** ✅

## 🛠 Desarrollo local

```bash
npm install
npm run dev
```

## ✏️ Personalizar links

Edita el array `LINKS` en `src/App.jsx` y cambia los `href: "#"` por tus URLs reales.

Para cambiar la foto, reemplaza el emoji 👨‍💻 por una etiqueta `<img>`.
